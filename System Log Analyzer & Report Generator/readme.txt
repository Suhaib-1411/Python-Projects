 OBJECTIVE

A CLI-based Python application developed to analyze system/application 
log files, detect log levels (INFO, WARNING, ERROR) using regular 
expressions (regex), and generate structured summary reports.

 FEATURES IMPLEMENTED

1. Load Log File: Validates and reads .txt/.log files line by line with 
   robust error handling (file not found/empty file checks).
2. Analyze Logs: Utilizes case-insensitive regex patterns to fetch 
   and count occurrences of INFO, WARNING, and ERROR logs.
3. Search & Filter: Allows dynamic keyword filtering across log contents.
4. Report Generation: Exports a cleanly formatted file named 'report.txt'.

 FOLDER CONTENTS

* main.py          - The core Python script containing the application logic.
* Sample_log.txt   - The dummy system log dataset used for testing.
* report.txt       - The final summary output generated by the script.
* demo_video.mp4   - A short 1-minute walkthrough demonstrating runtime usage.
* README.txt       - This documentation file.

 HOW TO RUN THE APPLICATION

1. Ensure Python 3.x is installed on your operating system.
2. Open VS Code and direct your workspace to the Task 1 folder.
3. Open the integrated terminal inside VS Code (Terminal -> New Terminal).
4. Run the script using the following command:
   
   python main.py

5. Interact with the application using the CLI menu choices (1-5):
   - Choose '1' and input 'Sample_log.txt' to load the dataset.
   - Choose '2' to instantly calculate the log summaries.
   - Choose '3' to run custom keyword string searches.
   - Choose '4' to write the results safely into 'report.txt'.
   - Choose '5' to cleanly terminate the runtime loop.
